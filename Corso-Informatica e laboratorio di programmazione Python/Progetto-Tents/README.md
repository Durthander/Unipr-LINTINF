## Comandi

| Tasto | Azione |
|-------|--------|
| **Click sinistro** | Ruota la cella: vuoto → prato → tenda → vuoto |
| **G** | Mette automaticamente il prato dove è certo |
| **T** | Mette automaticamente le tende dove è certo |
| **A** | Suggerisce una mossa sicura |
| **R** | Resetta la partita |
| **V** | Risolve automaticamente il puzzle, (o almeno prova a risolverlo) |

## Simboli o Emoji (su mac le emoji non funzionano, quindi ho usato i simboli, ma visto che con le emoji il gioco è più bello visivamente, ho fatto una versione con emoji)

|---------|-------------|
| **T/🌳** | Albero |
| **X/⛺** | Tenda |
| **-/🟩** | Prato (cella vuota sicura) |
